# minha configuração para o neovim
  eu viz com muito amor

  ### o que faz?
    UX melhorada com o tema catppuccin, status bar bonita com o slimline, tabs com o nvim-cokeline,
    indicadores de git com o gitsigns, notificações bonitas e simples com o notifier,
    render-markdown para renderizar o arquivo markdown

## requisitos
  - Neovim ( a cima da versão 0.9 )
  - git ( instalar o repositório e plugins de integração do git )
  - extenção do navegador GhostText ( opcional para integrar com o navegador )
  - nerd fonte ( para melhorar o visual do neovim )

## instalação 
  
  * no linux e mac:
  ``` bash
    git clone https://github.com/ovodlol/my_configs.nvim.git ~/.config/nvim
  ```

  * no windows:
  ``` ps
      git clone https://github.com/ovodlol/my_configs.nvim.git "$env:LOCALAPPDATA\nvim"  
  ```
